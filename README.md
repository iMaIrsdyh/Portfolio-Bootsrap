# **Personal Portfolio Website**

A modern and responsive **personal portfolio website** built using **HTML5, Bootstrap 5, and Vanilla JavaScript**. This project is designed to showcase personal information, services, projects, and contact details with a clean and professional UI.

---

## **Project Purpose**

This project was created to:

- Demonstrate responsive web design using **Bootstrap 5**
- Apply structured layout and reusable UI components
- Implement interactivity using **Vanilla JavaScript**
- Serve as a **professional portfolio website** for personal branding

---

## **Key Features**

- **Responsive & mobile-friendly layout**
- **Multi-page navigation** (Home, About, Projects, Contact)
- **Hero section** with introduction and call-to-action
- **Services section** using card components
- **Testimonials carousel** (Bootstrap Carousel)
- **About section** with social media integration
- **Custom CSS styling**
- **JavaScript-based UI interactions**

---

## **Tech Stack**

### **Frontend**

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Bootstrap Icons**
- **JavaScript (ES6 / Vanilla JS)**

---

## **Implementation Details**

- Layout is built using **Bootstrap 5 Grid System & Utility Classes**
- Components such as **Navbar, Cards, Buttons, and Carousel** use Bootstrap components
- Custom UI styling is handled in **style.css**
- Interactive behavior and animations are implemented in **javascript.js**
- Code follows **separation of concerns** (HTML, CSS, JavaScript)
